# Web Site Dra. Isabelle Climaco

Este é o site institucional premium para a Dra. Isabelle Climaco, desenvolvido com foco em elegância, autoridade e conversão.

## Tecnologias Utilizadas

- **HTML5 & CSS3**
- **Tailwind CSS**: Para estilização rápida e responsiva.
- **Google Fonts**: Fontes 'Playfair Display' (serifa elegante) e 'Montserrat' (moderna e legível).
- **Lucide Icons**: Conjunto de ícones minimalistas e modernos.
- **AOS (Animate On Scroll)**: Biblioteca para animações fluidas ao rolar a página.

## Estrutura do Projeto

- `index.html`: Arquivo principal contendo toda a estrutura, estilos customizados e lógica de animações.

## Como Visualizar

Basta abrir o arquivo `index.html` em qualquer navegador moderno.

## Observações Importantes

Os dados de contato (Telefone, Email, WhatsApp) e o número da OAB no rodapé são espaços reservados (placeholders). Recomenda-se atualizá-los com as informações reais antes da publicação.
